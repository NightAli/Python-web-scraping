# Python-web-scraping

範例與練習的網路爬蟲程式集合（包含 Selenium、BeautifulSoup、pandas 等）。

快速開始

1. 安裝必要套件：

```powershell
pip install -r requirements.txt
```

2. 以 Selenium 執行的程式需準備 chromedriver，並放在與程式相同資料夾或 PATH 可取得的位置。

注意事項

- 請勿把 Personal Access Token（PAT）或密碼加入版本控制。若曾誤貼請立即 revoke。 
- 部分資料檔（.xlsx、.jpg）已加入 repository，如需忽略未來相似檔案請參考 `.gitignore`。

主要目錄

- `selenium/`：Selenium 範例與練習檔。
- `mobile01/`, `tvbs/`, `weeknews/` 等：各網站爬蟲範例。
